# プロジェクト概要
## プロジェクト名
`discord_LLM_news`

## プロジェクトの目的
```
RSSを取得し、まとめ、それをLLMに投げ、いい感じに要約してもらう。
最終的にはdiscordに送信するまでを目標とする。
へっへがpythonで作っていたため、対抗してRustで作成する。
```


## プロジェクト構成
本プロジェクトは、バックエンドだけの簡潔なつくりとなっている。
- `backend/`: Rust (Cargo)

# ビルド&実行の備忘録
## バックエンド側 (Backend)
基本的に開発用を使ってればok
```bash
# 開発中に使うコマンド
cargo run

# ビルドだけしたいとき
# 後で立ち上げたいとき
cargo build

# 本番用(リリースビルド)
# 最適化を最大限に効かせる
cargo build --release
```
