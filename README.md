### ライブラリの依存関係？メモ
main.rsとlib.rsはすごい特殊な関係らしい。<br>
同じ階層にいるconfig.rsとではuseの書き方が違う。<br>
簡単にメモると次のようになる
```
・lib.rs
ライブラリの定義を行うとこであるため、
mod パッケージ名
または
pub mod パッケージ名
を宣言し、使うときは
use crate::パッケージ名

・main.rs
ライブラリの定義は行っているはずである。
しかし、crateは使えないらしいため、
use プロジェクト名::パッケージ名

・config.rs
完全に子らしい。そのため、
use crate::パッケージ名

```
